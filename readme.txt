== Description ==

With WP Logo Grid logos can be displayed with a shortcode in a clear and stylish way.
Insert the following shortcode anywhere you want in a post, a page, or a text widget and add IDs of the desired logo images within x-shortcode tags, e.g.:

## [x]47,264,193,132,174,453[/x]


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Insert the following opening and closing shortcodes anywhere in a post, a page, or a text widget: `[x]...[/x]`
4. Instead of '...' you enter the IDs of the images that you want to display in the logo grid, e.g.: `[x]47,264,193,132,174,453[/x]`
